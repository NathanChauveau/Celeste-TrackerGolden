# Celeste-TrackerGolden
The goal of this project is to give a way to track your runs during an attemp of a golden strawberry on Celeste

This project will take quite a lot of time, it is also my 1st personal project that i'll do completely alone.
It will serve as a traning for me :)

## Functionnality 
You'll be able to :
- Create your account
- Save a run with the time and total of death
- Delete runs you deemed not useful
- See which run are the best on time, low death, and in both category
- You'll be able to add your own map by adding the image of each room
- Ping a room on the map where you went the furthest with the golden berry
